GeoPhy - Data converter tool

GeoPhy is a tool built in WinForms to edit raw GPS data from geophysical survey acquisitions (i.e. Resistivity and Seismic Refraction) into an AutoCAD format. Built to optimize the process of editing data before inputting them in autoCAD. Still in progress the export phase of the CSV file to .dxf file.

UTriSS - Tic Tac Toe Game

UTriSS is a Tic Tac Toe Game version built using Xamarin.Android. It was first built in console as an exercise for practicing C#, then I decided to built an Andoid App taking advantage of Xamarin.Android that it allows to built android app written in C#. You can view the code in GitHub or download the .apk file to test it on your devices.

Pong Game

Back in the days when Pong was a famous desktop game. The game is a two players game. If you press View Code button you can check the code and if you press run you can enjoy the actual game. The game has been built in Python for an assignment of Interactive Python course in Coursera.

StopWatch Game

StopWatch is a game built in Python for an Interactive Python course in Coursera. It's simple game that consists in proving the capacity of stopping the watch at :00. The app will count how many times you stop the watch on time over the total attempts.
